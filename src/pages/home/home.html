Intro

Firebase in Angular 2

If you're interested in the files that I've changed to integrate Firebase and get this simple auth example working, they're:

  • www/index.html - you'll find that I've added the CDN reference to Firebase, as well as initialized the database
  • app/data/data.service.ts - sets up Firebase database references
  • app/data/user.service.ts - abstracts Firebase auth methods
  • app/pages/home/ - the currently active component
    • home.html - the template you're currently viewing
  • app/pages/basic-demo/ - basic database demo
    • basic-demo.html - sets up the template for displaying synced data
    • basic-demo.ts - builds out the @Page for displaying synced data
  • app/pages/auth/ - Firebase authentication demo
    • auth.html - sets up the template for displaying user auth states
    • auth.ts - builds out the @Page for integrating user auth states
    • components/ - components used for interacting with auth states
      • login.component.ts - builds the login form, integrats the user service login function
      • logout.component.ts - allows the user to logout, resetting the view to display the login form
  • app/pages/tabs/tabs.html - revised tab titles
